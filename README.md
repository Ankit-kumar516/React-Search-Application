Mini Project: Search Application
Data Initialization: Build a component that fetches a user list from the JSON Placeholder API, specifically when the component first mounts.
State Management: Implement state logic to store the fetched users and a "Loading..." message that displays only until the data is successfully received.
Dynamic Filtering: Create a controlled search input that filters the displayed list of names in real-time, ensuring the search is case-insensitive.
Performance Optimization: As a bonus, implement a debounce function to limit how often the filtering logic runs while the user is actively typing.
